# Игра "Городки" (SFML 2.6.2)

Простая аркадная игра "Городки" на C++ с использованием библиотеки SFML.

## Описание проекта

Классическая игра "Городки", где игрок бросает биту, стараясь выбить все фигуры (городки) с поля.

### Структура проекта

| Файл                    | Назначение |
|------------------------|----------|
| `main.cpp`             | Главный файл. Создаёт окно, музыку и запускает StateManager |
| `State.h`              | Базовый класс State и StateManager (машина состояний) |
| `MenuState.h/.cpp`     | Главное меню с выбором (Play, Rules, Settings, Exit) |
| `GameState.h/.cpp`     | Основная игра: бита, городки, физика броска, уровни |
| `RulesState.h/.cpp`    | Экран с правилами игры |
| `SettingsState.h/.cpp` | Настройки: звук и выбор разрешения экрана |
| `GameConfig.h`         | Структура с глобальными настройками игры |
| `media/`               | Папка с шрифтом (`arial.ttf`) и музыкой (`all_mix.mp3`) |

### Как играть

- **Стрелки ↑ ↓** — навигация по меню
- **Enter** — подтверждение выбора
- **ESC** — возврат в меню из любого экрана
- В игре:
  - **Space** (удерживать) — заряд силы броска
  - **← →** — изменение угла броска
  - **1, 2, 3** — выбор уровня
  - **R** — перезапуск уровня
  - **M** — включить/выключить музыку

### Управление настройками

В разделе **Settings**:
- **Left / Right** — переключение звука и разрешения экрана
- **Enter** — сохранить и вернуться в меню

### Требования

- SFML 2.6.2
- Компиляция на macOS (Apple Silicon):

```bash
g++ -std=c++17 -o gorodki main.cpp MenuState.cpp GameState.cpp RulesState.cpp SettingsState.cpp \
  -I/opt/homebrew/include \
  -L/opt/homebrew/lib \
  -lsfml-graphics -lsfml-window -lsfml-system -lsfml-audio -Wno-unused-result
